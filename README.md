# RentVago

## Objetivo

- Nuevo repositorio para practicar para el examen, sabiendo que el examen es una integracion de funciones de nuestro proyecto.

## Funciones actuales

- Coneccion a Supabase 
- Login y Register, funcionando con JWT, ZOD y contrasenas Hasheadas.
- JWT Guardada en las cookies

## Configuracion rapida 

1. `bun install`
2. `bun dev`
3. Para la base de datos: `bunx prisma db push`

## Scrapping

- Fastify y cheerio para obtener datos de propiedades.
- Usando scraper-api.ts para obtener datos de propiedades.
